Multi-Zone Graph Layout

How should multiple temperature readings appear on the firing chart?

A

Single Graph, Overlaid Traces

All zone temperatures as colored lines on one shared Y-axis. One target profile line. Compact, easy to compare zones at a glance. Legend identifies each zone.

B

Stacked Subplots

Each zone gets its own mini-graph stacked vertically, sharing the same X (time) axis. More space per zone, but takes more screen real estate. Harder to visually compare zones.

C

Single Graph + Zone Summary Panel

Overlaid traces like A, but with a sidebar/panel showing each zone's current temp, target, deviation, and heat output as cards. Best of both worlds — the graph stays clean while the panel gives per-zone detail.
